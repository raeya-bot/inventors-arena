# The Inventors Arena — Website Revamp

A premium redesign of [theinventorsarena.com](https://theinventorsarena.com).
Single-file build: `index.html` with embedded CSS + JS (no build step required).

**Design direction:** gladiatorial colosseum meets modern venture capital firm —
dark, commanding, polished. Cinzel / Raleway / Inter, Arena Gold accents,
3D badge logo, triangle watermark motifs, scroll-reveal animations.

## Run locally
Open `index.html` in a browser, or:
```
npx serve .
```

## Deploy
Static site — deployable to Vercel/Netlify/any static host with zero config.
